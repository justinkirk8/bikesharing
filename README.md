# bikesharing

## Overview of Analysis:
Client wants to start a bike share buisness. Requested an analysis of the data reported by CitiBike for the month of August in 2019. Jupyter Notebook was used to format the data and Tableau was used to analyze and visualize the data.

## Resources
- Software: Python, Jupyter Notebook, Tableau
- Source Data: 201908-citibike-tripdata.csv from https://ride.citibikenyc.com/system-data
